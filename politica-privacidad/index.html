Política de privacidad

1. Responsable del tratamiento

En cumplimiento de la normativa vigente en materia de protección de datos personales, se informa al usuario de que los datos personales facilitados a través del sitio web tallerdedigitalizacion.com serán tratados por:

Responsable: Pablo Leone
NIF: 41675399C
Domicilio: Numancia 4, Fuensalida, España
Email de contacto: info@tallerdedigitalizacion.com
Sitio web: tallerdedigitalizacion.com

En adelante, el responsable podrá ser denominado como “el titular”, “el responsable” o “Taller de Digitalización”.

2. Finalidad de esta política

La presente Política de Privacidad tiene como finalidad informar de forma clara sobre cómo se recogen, utilizan, conservan y protegen los datos personales de los usuarios que acceden al sitio web, contactan con el titular, reservan una llamada, completan formularios, realizan autoevaluaciones, descargan materiales o interactúan con herramientas externas integradas en la web.

3. Datos personales que pueden tratarse

En función del uso que el usuario haga del sitio web, podrán tratarse las siguientes categorías de datos:

  • Datos identificativos: nombre, apellidos, cargo o empresa.
  • Datos de contacto: email, teléfono, WhatsApp u otros medios facilitados por el usuario.
  • Datos profesionales: nombre de empresa, sector, tamaño aproximado, situación operativa, necesidades o comentarios enviados voluntariamente.
  • Datos introducidos en formularios o autoevaluaciones: respuestas, puntuaciones, observaciones y resultados generados.
  • Datos técnicos: dirección IP, identificadores de cookies, navegador, dispositivo, páginas visitadas, fecha y hora de acceso, origen de la visita y comportamiento de navegación, cuando proceda.
  • Datos derivados de comunicaciones: mensajes enviados por formularios, email, WhatsApp, Calendly, Chatbase u otros canales habilitados.

No se solicita intencionadamente información especialmente protegida, como datos de salud, ideología, religión, afiliación sindical, orientación sexual, datos biométricos o similares. El usuario debe abstenerse de facilitar este tipo de información a través de los formularios, chatbot o canales de contacto.

4. Finalidades del tratamiento

Los datos personales podrán ser tratados para las siguientes finalidades:

4.1. Atender consultas y solicitudes de contacto

Se tratarán los datos necesarios para responder a mensajes enviados por formularios, email, WhatsApp, Calendly, Chatbase u otros canales habilitados.

4.2. Gestionar reservas de llamadas

Si el usuario reserva una llamada gratuita de 30 minutos mediante Calendly, se tratarán los datos necesarios para gestionar la solicitud, confirmar la cita, enviar recordatorios y mantener la comunicación relacionada con dicha reunión.

4.3. Realizar autoevaluaciones y generar resultados

Si el usuario completa la Autoevaluación del Impuesto Operativo u otras herramientas similares, se tratarán las respuestas introducidas para calcular puntuaciones, mostrar resultados, generar una interpretación inicial y, en su caso, permitir la descarga de un diagnóstico o informe.

La autoevaluación tiene carácter orientativo y no sustituye un diagnóstico profesional completo.

4.4. Gestionar formularios personalizados

Los formularios del sitio web son personalizados y pueden enviar la información introducida a un backend propio o controlado por el responsable.

Los datos enviados mediante dichos formularios se utilizarán únicamente para la finalidad indicada en cada caso: contacto, descarga de materiales, autoevaluación, solicitud de información, reserva de llamada o seguimiento comercial consentido.

4.5. Enviar materiales descargables

Si el usuario solicita descargar el método, informes, guías, documentos o materiales similares, se tratarán los datos facilitados para entregar el recurso solicitado y registrar la solicitud.

4.6. Enviar comunicaciones comerciales o informativas

Actualmente no se utiliza una herramienta externa de email marketing.

Solo se enviarán comunicaciones comerciales, newsletters o contenidos informativos cuando exista una base legal para ello, como el consentimiento previo del usuario o una relación contractual previa que lo permita.

El usuario podrá darse de baja o retirar su consentimiento en cualquier momento escribiendo a info@tallerdedigitalizacion.com.

4.7. Analizar el uso del sitio web

Podrán tratarse datos técnicos y de navegación mediante Google Analytics, siempre que el usuario haya prestado su consentimiento cuando sea necesario.

Estos datos permiten entender el uso de la web, mejorar contenidos, detectar problemas técnicos y medir la eficacia de campañas, formularios o llamadas a la acción.

4.8. Ofrecer asistencia mediante chatbot

El sitio web utiliza Chatbase como herramienta de chatbot o asistente automatizado. Esta herramienta permite al usuario realizar preguntas sobre los servicios, metodología, precios, funcionamiento de la web u otros contenidos disponibles.

Los mensajes introducidos por el usuario en el chatbot podrán ser tratados para generar respuestas y mantener el funcionamiento de la herramienta.

El usuario no debe introducir información confidencial, sensible o innecesaria en el chatbot.

4.9. Mantener la seguridad y funcionamiento del sitio web

Se podrán tratar datos técnicos estrictamente necesarios para garantizar la seguridad, disponibilidad, prevención de abusos, detección de errores y correcto funcionamiento del sitio web.

5. Base legal del tratamiento

Las bases legales que legitiman el tratamiento de datos son:

  • Consentimiento del usuario: para formularios, descarga de materiales, suscripción a comunicaciones, cookies analíticas, uso de herramientas no estrictamente necesarias o aceptación de finalidades específicas.
  • Aplicación de medidas precontractuales: cuando el usuario solicita información, una llamada, un diagnóstico o una propuesta de servicios.
  • Ejecución de un contrato: cuando exista contratación de servicios.
  • Interés legítimo: para mantener la seguridad del sitio web, prevenir abusos, responder a comunicaciones iniciadas por el usuario o conservar evidencias básicas de la relación.
  • Cumplimiento de obligaciones legales: cuando sea necesario conservar determinada información por motivos fiscales, contables, legales o administrativos.

6. Conservación de los datos

Los datos personales se conservarán durante el tiempo necesario para cumplir la finalidad para la que fueron recogidos.

Con carácter general:

  • Los datos de contacto se conservarán mientras exista una relación, solicitud abierta o interés legítimo en mantener la comunicación.
  • Los datos de consultas se conservarán durante el tiempo necesario para responder y gestionar la solicitud.
  • Los datos relacionados con servicios contratados se conservarán durante la relación contractual y, posteriormente, durante los plazos legales aplicables.
  • Los datos asociados a comunicaciones comerciales se conservarán hasta que el usuario retire su consentimiento o solicite la baja.
  • Los datos de autoevaluaciones podrán conservarse mientras sean necesarios para generar el resultado, facilitar seguimiento o atender la solicitud del usuario.
  • Los datos técnicos y cookies se conservarán según lo indicado en la Política de Cookies.
  • Las conversaciones mantenidas a través de Chatbase podrán conservarse conforme a la configuración del servicio y a su política aplicable.

Una vez finalizados los plazos aplicables, los datos serán eliminados o bloqueados cuando proceda.

7. Destinatarios y encargados del tratamiento

No se venderán datos personales a terceros.

No obstante, para prestar los servicios y mantener el funcionamiento del sitio web, podrán intervenir proveedores externos que traten datos por cuenta del responsable, como:

  • Proveedores de hosting y servicios web.
  • Backend propio o proveedor técnico usado para procesar formularios personalizados.
  • Calendly, para la reserva de llamadas.
  • Google Analytics, para analítica web si el usuario acepta cookies analíticas.
  • Chatbase, para el chatbot o asistente automatizado.
  • WhatsApp, para contacto directo mediante mensajería.
  • Linktree, como hub externo de enlaces.
  • Proveedores de email o infraestructura técnica necesarios para responder comunicaciones.

Cada proveedor podrá tratar datos conforme a sus propias condiciones y políticas de privacidad. Cuando el usuario interactúe con servicios externos, deberá revisar sus respectivas políticas.

8. Transferencias internacionales de datos

Algunos proveedores tecnológicos pueden estar ubicados fuera del Espacio Económico Europeo o utilizar infraestructuras internacionales.

En caso de producirse transferencias internacionales de datos, se procurará que estén amparadas por garantías adecuadas conforme al RGPD, como cláusulas contractuales tipo, decisiones de adecuación u otros mecanismos legalmente previstos.

El uso de servicios como Google Analytics, Calendly, Chatbase, WhatsApp o Linktree puede implicar tratamiento de datos fuera del Espacio Económico Europeo conforme a las condiciones de dichos proveedores.

9. Derechos del usuario

El usuario puede ejercer los siguientes derechos en materia de protección de datos:

  • Acceso: saber qué datos personales se están tratando.
  • Rectificación: solicitar la corrección de datos inexactos.
  • Supresión: solicitar la eliminación de datos cuando proceda.
  • Oposición: oponerse a determinados tratamientos.
  • Limitación del tratamiento: solicitar que se limite el uso de los datos.
  • Portabilidad: recibir los datos en formato estructurado cuando sea aplicable.
  • Retirada del consentimiento: retirar el consentimiento otorgado en cualquier momento.

Para ejercer estos derechos, el usuario puede escribir a:

info@tallerdedigitalizacion.com

La solicitud deberá indicar claramente el derecho que se desea ejercer y podrá requerirse información adicional para verificar la identidad del solicitante.

El usuario también tiene derecho a presentar una reclamación ante la Agencia Española de Protección de Datos si considera que el tratamiento de sus datos no se ajusta a la normativa aplicable.

10. Seguridad de los datos

El responsable aplicará medidas técnicas y organizativas razonables para proteger los datos personales frente a pérdida, uso indebido, acceso no autorizado, alteración, divulgación o destrucción.

No obstante, ningún sistema conectado a Internet puede garantizar una seguridad absoluta. El usuario debe utilizar el sitio web y los canales de contacto de forma prudente, evitando enviar información sensible o innecesaria.

11. Formularios del sitio web

Los formularios disponibles en el sitio web podrán solicitar datos necesarios para responder consultas, gestionar reservas, enviar materiales, realizar autoevaluaciones o mantener comunicaciones.

Los campos marcados como obligatorios serán necesarios para atender la solicitud correspondiente. Si el usuario no facilita esos datos, es posible que no se pueda gestionar su petición.

Al enviar un formulario, el usuario declara que los datos facilitados son veraces, actualizados y propios, o que cuenta con autorización suficiente para comunicarlos.

12. Autoevaluación del Impuesto Operativo

La web puede incluir una autoevaluación basada en preguntas sobre áreas operativas de la empresa, como dirección, procesos, soporte, operación técnica, identidad digital, datos, trazabilidad, seguridad y automatización.

Las respuestas introducidas podrán utilizarse para calcular un score orientativo, mostrar áreas débiles, generar resultados descargables y proponer una conversación posterior.

La autoevaluación no constituye asesoramiento profesional personalizado ni garantiza resultados. Su finalidad es ofrecer una primera lectura del estado operativo percibido por el usuario.

13. Comunicaciones por WhatsApp

Si el usuario contacta a través de WhatsApp, sus datos serán tratados para gestionar la conversación y responder a su solicitud.

El uso de WhatsApp implica que el usuario interactúa con una plataforma externa, sujeta a sus propias condiciones y política de privacidad.

Se recomienda no enviar por WhatsApp información confidencial, sensible o innecesaria.

14. Reservas mediante Calendly

Si el usuario reserva una llamada mediante Calendly, los datos introducidos serán tratados para gestionar la cita, enviar confirmaciones, facilitar recordatorios y mantener la comunicación relacionada con la reunión.

Calendly actúa como proveedor externo y puede tratar datos conforme a sus propias condiciones y política de privacidad.

El usuario debe revisar también las condiciones y política de privacidad de Calendly antes de utilizar la herramienta.

15. Chatbot mediante Chatbase

El sitio web incorpora Chatbase como herramienta de chatbot o asistente automatizado para responder preguntas frecuentes o facilitar información sobre los servicios.

Los mensajes introducidos por el usuario podrán ser tratados por Chatbase para generar respuestas y mantener el funcionamiento de la herramienta.

El usuario debe evitar introducir datos personales innecesarios, información confidencial, secretos empresariales, credenciales, datos sensibles o información de terceros en el chatbot.

16. Cookies y tecnologías similares

Este sitio web puede utilizar cookies técnicas necesarias para su funcionamiento y, previo consentimiento cuando sea exigible, cookies analíticas o de terceros.

El uso de cookies se regula en la Política de Cookies, donde el usuario puede consultar información detallada sobre tipos de cookies, finalidades, duración, proveedores y forma de configurar o retirar el consentimiento.

17. Menores de edad

Los servicios de este sitio web están dirigidos a profesionales, empresas y personas mayores de edad.

No se recogen intencionadamente datos de menores. Si el responsable detecta que se han recibido datos de un menor sin autorización válida, procederá a eliminarlos cuando corresponda.

18. Exactitud de los datos

El usuario se compromete a facilitar datos veraces, exactos y actualizados.

El responsable no será responsable de consecuencias derivadas de datos falsos, incompletos, inexactos o desactualizados facilitados por el usuario.

19. Cambios en la política de privacidad

El responsable podrá modificar esta Política de Privacidad para adaptarla a cambios legales, técnicos, comerciales u operativos.

La versión vigente estará siempre disponible en el sitio web.
