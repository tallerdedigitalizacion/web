Política de cookies

1. ¿Qué son las cookies?

Las cookies son pequeños archivos que se descargan en el dispositivo del usuario al acceder a determinadas páginas web.

Las cookies permiten, entre otras cosas, que una página web funcione correctamente, recuerde determinadas preferencias, obtenga información estadística sobre la navegación o facilite la integración con servicios de terceros.

También pueden utilizarse tecnologías similares, como píxeles, etiquetas, almacenamiento local del navegador, identificadores o scripts de terceros. En esta política, el término “cookies” se utiliza de forma amplia para referirse también a estas tecnologías cuando resulte aplicable.

3. ¿Qué tipos de cookies utiliza este sitio web?

Este sitio web puede utilizar cookies propias y de terceros con distintas finalidades.

3.1. Cookies técnicas o necesarias

Son aquellas necesarias para que la web funcione correctamente y pueda prestar los servicios solicitados por el usuario.

Pueden utilizarse, por ejemplo, para:

  • permitir la navegación por la web
  • recordar preferencias técnicas
  • mantener la seguridad del sitio
  • gestionar formularios
  • recordar ajustes básicos
  • permitir el funcionamiento de elementos interactivos

Estas cookies no requieren consentimiento cuando son estrictamente necesarias para prestar un servicio solicitado por el usuario o para el funcionamiento básico del sitio.

3.2. Cookies de análisis o medición

Son cookies que permiten obtener información sobre cómo los usuarios utilizan la web, qué páginas visitan, cuánto tiempo permanecen, desde dónde llegan o qué acciones realizan.

Este sitio web puede utilizar Google Analytics 4 para analizar el uso de la web y mejorar sus contenidos, navegación, rendimiento y llamadas a la acción.

La información recogida puede incluir datos técnicos como dirección IP, navegador, dispositivo, páginas visitadas, eventos de interacción, fecha, hora y origen de la visita.

3.3. Cookies de terceros

Este sitio web puede integrar o enlazar servicios de terceros que pueden instalar cookies o tecnologías similares cuando el usuario interactúa con ellos.

Entre estos servicios pueden estar:

  • Google Analytics, para analítica web.
  • Calendly, para la reserva de llamadas.
  • Chatbase, para el chatbot o asistente automatizado.
  • WhatsApp, para facilitar el contacto directo.
  • Linktree, como hub externo de enlaces.
  • Servicios técnicos vinculados a formularios personalizados o backend propio.

Cada uno de estos servicios puede utilizar sus propias cookies y tratar datos conforme a sus propias políticas.

Google puede tratar datos en servidores ubicados fuera del Espacio Económico Europeo. El tratamiento de datos por parte de Google se rige por sus propias condiciones y políticas.

6. Calendly

Este sitio web puede enlazar o integrar Calendly para permitir la reserva de llamadas gratuitas de 30 minutos.

Cuando el usuario utiliza Calendly, puede introducir datos como nombre, email, empresa, cargo, disponibilidad, comentarios u otra información necesaria para gestionar la reunión.

Calendly puede utilizar cookies propias y de terceros para prestar su servicio, recordar preferencias, gestionar sesiones o analizar el uso de su plataforma.

El uso de Calendly se rige también por sus propias condiciones y política de privacidad.

7. Chatbase

Este sitio web utiliza Chatbase como chatbot o asistente automatizado.

Chatbase puede utilizar cookies o tecnologías similares para permitir el funcionamiento del chatbot, mantener conversaciones, analizar interacciones o mejorar el servicio.

El usuario debe evitar introducir en el chatbot información sensible, confidencial, credenciales, secretos empresariales, datos de terceros o información innecesaria.

El uso de Chatbase se rige también por sus propias condiciones y política de privacidad.

8. WhatsApp y Linktree

Este sitio web puede incluir enlaces o botones hacia WhatsApp y Linktree.

Al hacer clic en estos enlaces, el usuario abandona el sitio web y accede a servicios externos, que pueden utilizar sus propias cookies o tecnologías similares.

El titular de esta web no controla las cookies instaladas por dichos terceros ni sus políticas de tratamiento de datos.

9. Consentimiento

Las cookies técnicas necesarias pueden utilizarse sin consentimiento previo, al ser necesarias para el funcionamiento básico del sitio web.

Las cookies analíticas, publicitarias o de terceros que no sean estrictamente necesarias requieren, con carácter general, el consentimiento previo del usuario conforme a la normativa aplicable.

El sitio web podrá mostrar un banner o panel de configuración para que el usuario pueda aceptar, rechazar o configurar las cookies no necesarias.

Importante: si la configuración técnica actual de la web no permite bloquear cookies no necesarias antes del consentimiento, deberá adaptarse la implementación técnica para que el uso de cookies analíticas o de terceros se ajuste correctamente a la normativa aplicable.

10. Cómo configurar o desactivar cookies desde el navegador

El usuario puede permitir, bloquear o eliminar las cookies instaladas en su dispositivo mediante la configuración del navegador que utilice.

A continuación se indican enlaces de ayuda de los navegadores más habituales:

  • Google Chrome: configuración de privacidad y seguridad del navegador.
  • Mozilla Firefox: opciones de privacidad y seguridad.
  • Safari: preferencias de privacidad.
  • Microsoft Edge: configuración de cookies y permisos del sitio.

El bloqueo de determinadas cookies puede afectar al funcionamiento de algunas funcionalidades de la web o de servicios externos integrados.

11. Retirada del consentimiento

Cuando las cookies se basen en el consentimiento del usuario, este podrá retirarlo en cualquier momento mediante el panel de configuración de cookies, si está disponible, o modificando la configuración de su navegador.

La retirada del consentimiento no afectará a la licitud del tratamiento realizado con anterioridad.

12. Transferencias internacionales

El uso de servicios de terceros como Google Analytics, Calendly, Chatbase, WhatsApp o Linktree puede implicar transferencias internacionales de datos o tratamientos realizados fuera del Espacio Económico Europeo.

Dichos tratamientos se regirán por las garantías, condiciones y políticas de privacidad aplicables en cada proveedor.

13. Actualizaciones de esta política

Esta Política de Cookies puede modificarse cuando se produzcan cambios en la configuración de cookies, servicios utilizados, normativa aplicable o criterios de las autoridades competentes.

Se recomienda revisar periódicamente esta política.
